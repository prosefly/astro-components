Use `@prosefly/astro-components/markdown` when a project wants the markdown
enhancements without the full Lotus theme.

## Astro Configuration

```ts title="astro.config.ts"
import { defineConfig } from 'astro/config';
import {
  rehypeImageGallery,
  remarkPackageManagerTabs,
} from '@prosefly/astro-components/markdown';

export default defineConfig({
  markdown: {
    remarkPlugins: [remarkPackageManagerTabs],
    rehypePlugins: [rehypeImageGallery],
  },
});
```

## Package Manager Tabs

`remarkPackageManagerTabs` recognizes common install and run commands and can
generate tabs for package manager alternatives.

```sh
npm install @prosefly/astro-components
```

```sh
pnpm add @prosefly/astro-components
```

```sh
yarn add @prosefly/astro-components
```

## Image Galleries

`rehypeImageGallery` turns paragraphs that contain only images into gallery
figures. Import the runtime once in the page shell to load styles and enable
previous and next controls.

```astro
---
import '@prosefly/astro-components/markdown/image-gallery.css';
---

<script>
  import '@prosefly/astro-components/markdown/image-gallery.js';
</script>
```

<Callout type="note" title="Lotus default">
  Lotus enables both transforms by default. Configure them directly only when
  using this package without Lotus.
</Callout>
